Access

W3 has produced Web Content Accessibility Guidelines in order to make web content more accessible for different people. There are various ways to improve accessibility in web designs including the following:

Improve access for text

  • Webpage has title that describes topic/purpose
  • Use plain language for text, without jargon, idioms, slang, sarcasm, or other complicated terms. Reading level should be lower secondary level.
  • Do not justify text and provide ability to resize up to 200%
  • Do not center align large blocks of text
  • Use decorative fonts sparingly
  • Use one h1 heading per page and structure headings in logical sequence
  • Headings should be descriptive of topic or purpose
  • Use line spacing: at least 1.5 within paragraphs and space between paragraphs is at least 1.5 larger than line spacing

Improve access in interaction

  • Make interactive aspects distinguishable
  • Provide context for links and buttons (don't just say "Click Here")
  • Don't just rely on shape or colour to indicate interaction

Improve access for visuals

  • Use alt text to provide image descriptions
  • Test site on different device formats
  • Select color combinations with at least 4.5:1 contrast: Adobe Color Contrast Checker
  • Test color contrast and perform accessibility checks: guide for Firefox and guide for Chrome

More documents from W3

  • Full Web Content Accessibility Guidelines (WCAG) 2.0
  • How to Meet WCAG (Quick Reference)

Notes on saving files on Mac Lab Computers

Files saved locally to a Mac will only be available on that Mac. So do not save files to the documents folder as previously indicated.

Ways to save:

  1. Your computer science accounts have a certain amount of storage they can save to, which can be accessed on any machine. This shows as a Network Drive (a little earth icon at the right of the dock).
  2. You can also use any file sharing software e.g., onedrive, google drive, dropbox, etc. You all have access to an Office365 student license which includes OneDrive.
  3. You can also email the file to yourself. Just write an email to your own address and attach the file to this email.
  4. You can save to a USB or external hard drive, but note that the Windows file format NTFS will not be writable to from a Mac. Your drive will need to be exFAT format to work across Windows and Mac.
