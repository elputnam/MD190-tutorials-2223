Hello World!

Now is it is time to make your first website!

  1. Create a folder in your computer science account drive (see below notes on saving).
  2. Open up Visual Studio Code (VSCode) in applications.
  3. Select "Open Folder" and open your folder.
  4. Create a new html document and save as index.html
  5. Copy this text and past into your document:

    6. <!DOCTYPE html>
    <html>

    <head>

    </body>

    </body>

    </html>

  6. Check out what your page looks like in the browser. Find the file in your document folder and double click to open in web browser.

Notes on saving files on Mac Lab Computers

Files saved locally to a Mac will only be available on that Mac. So do not save files to the documents folder as previously indicated.

Ways to save:

  1. Your computer science accounts have a certain amount of storage they can save to, which can be accessed on any machine. This shows as a Network Drive (a little earth icon at the right of the dock).
  2. You can also use any file sharing software e.g., onedrive, google drive, dropbox, etc. You all have access to an Office365 student license which includes OneDrive.
  3. You can also email the file to yourself. Just write an email to your own address and attach the file to this email.
  4. You can save to a USB or external hard drive, but note that the Windows file format NTFS will not be writable to from a Mac. Your drive will need to be exFAT format to work across Windows and Mac.

Use multiple of the above for saving anything important (ie. save to your Computer Science account AND OneDrive) ðŸ˜Š
