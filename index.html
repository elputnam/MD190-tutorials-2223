Elements of a page

HTML element (or simply, element) — a unit of content in an HTML document formed by HTML tags and the text or media it contains.
1. HTML Tag — the element name, surrounded by an opening (<) and closing (>) angle bracket.
2. Opening Tag — the first HTML tag used to start an HTML element. The tag type is surrounded by opening and closing angle brackets.
3. Content — The information (text or other elements) contained between the opening and closing tags of an HTML element.
4. Closing tag — the second HTML tag used to end an HTML element. Closing tags have a forward slash (/) inside of them, directly after the left angle bracket.

A more detailed reference can be found at The W3 Schools. Here you will find reference to different types of tags and what they do.

Hello World!

Now is it is time to make your first website!
1. Create a folder in your computer's documents section.
2. Open up Visual Studio Code (VSCode) in applications.
3. Select "Open Folder" and open your folder
4. Create a new html document and save as index.html
5. Copy this text and past into your document:

<!DOCTYPE html>
<html>
<body>

<h1>Hello World!</h1>
<p>Look at me code!</p>

</body>
</html>

Check out what your page looks like in the browser. Find the file in your document folder and double click to open in web browser.
