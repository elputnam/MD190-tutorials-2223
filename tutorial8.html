What is CSS?

CSS stands for Cascading Style Sheets, which are used to describe how HTML elements are to be presented. CSS uses one document to style a whole website, saving a lot of work and time. As noted in the lecture, CSS separates content (HTML) from presentation, as it is a separate document that informs the style of the site. Since first introduced in 1996, CSS has become a cornerstone of the web, with the capacity to include variations for site designs on different devices and screen sizes, layout websites, as well as introduce dynamic elements to static sites.

In short, CSS means that style is no longer in HTML (which it wasn't designed for anyway...).

For today's class, we will go over the creation of external style sheets, which you will start to do for your webpage. Then I will review some basic formatting and introduce the box model to begin page layout, which we will continue next week.

CSS syntax

A CSS ruleset (or simply rule) consists of a selector and a declaration block. Within the declaration block, there is a property and property value. As with HTML, syntax matters, so pay attention the use of curly brackets, spacing, colons, and semi-colons.

Image source: Mdn Web docs

Overview of terms:

  • selector is the HTML element(s) being styled
  • the declaration is the rule
  • the property is a way to style HTML element
  • the property value is how it is being styled

More on Selectors

CSS enables universal to precise changes to a website design, including styling all HTML elements or some, specific images, or even when a cursor hovers over a link.

Here are some selectors that may be useful for you when designing your site:

  • Element selector, tag, or type selector: all HTML elements of a specified type
  • ID selector: unique value assigned to element with specified ID
  • Class selector: creates a class for elements that can appear multiple times on same page
  • Pseudo-class selector: specified elements, but only in specific state like hovering or activated
  • Universal selector: applies to all elements; uses *

Box model

CSS is based on a model of boxes, where each HTML element is considered as wrapped in a box. Boxes are divided into the following properties: padding (space around content), border (solid line around padding), and margin (space around outside of border). Adjusting these qualities will alter the spacing around your elements. You can include one, two, three, or four values. Inputting one value will apply to all sides of the box; when inputting two, the first will affect top and bottom while the second will affect right and left. You can also adjust these values to remove the automatic styling of elements.
