Introduction to Google Analytics

For the first part of this week's tutorial, we are going to become familiar with Google analytics.

Please note that there are now two versions of Google Analytics, Google Analytics 4 (new version) and Universal Analytics (UA). Google plans to shut off UA on 1 July 2023.

Key Concepts

  • Metrics: quantitative numbers that measure data in counts, ratios, percentages, and so on.
  • Dimesions: qualitative categories that describe data in segments or breakouts.
  • Events: any interaction on a site (clicks, scrolls, product views, page views, ecommerce transactions, exceptions). Events are structured with a name, parameter, and value
  • Session: time frame of engagement; a visit to a site or app within a given time frame
  • Exit: leaving site or being inactive for at least 30 min (note: if user leaves site and comes back within 30 minutes, counts as original session)
  • User: visitor of site or app. Google identifies user by browser cookies, Google identity graph, or a USERID supplied by site. Same user can be tracked across multiple sessions or devices.
  • Conversion: measures if objectives are being reached; user completes an action or goal on website (ie. make purchase, click link, sign up for newsletter, spend time on site -- defined by organisation or business)
  • Source: where traffic came orginated from (ie. Instagram)
  • Medium: avenue traffic took to get from source to site (ie. social)
  • Attribution: what gets credit for user's journey to conversion; different touch points

Steps for getting started with Google Analytics

  1. Create a Google Analytics account by clicking the Start measuring button to sign up on the GA webpage.
  2. Select an Account name and check/uncheck data sharing settings.
  3. Under Property name, enter the name of your website or business (not your URL). Choose your time zone and currency. Then, click "Create".
  4. Once GA account is set up, click on Data Stream and select web.
  5. Input website URL, name stream, and decide whether to enable enhanced measurement
  6. In order to start process to install GA4 on site, scroll down to Google Tag and click view tag instructions
  7. Install GA4 using website builder or manually

We are going to shift our attention from the coding side of things to content creation, which includes an introduction to Adobe Photoshop. Photoshop is a raster/bitmap graphics editing software, created in 1988 by Thomas and John Knoll. It works through layering and masks to edit, collage, add text, and otherwise manipulate digital images.

In class, we are going to cover opening Photoshop, activating the device license, and becoming familiar with the interface. Please note that in order to access Photoshop, you will need to set up an account with Adobe. This can be done with your Maynooth University email address.
